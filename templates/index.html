About Project

How This Application Works

BART Abstractive

BART reads the input document and generates a new rewritten summary. It is suitable for English long-form legal or Terms and Conditions text.

TF-IDF Extractive

TF-IDF calculates word importance and selects the highest-scoring original sentences. It is used as a baseline model.

ROUGE Evaluation

ROUGE compares the generated summary with a reference summary using ROUGE-1, ROUGE-2, and ROUGE-L metrics.
